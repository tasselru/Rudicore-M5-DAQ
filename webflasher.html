Rudicore-M5 Web Installer

Install Rudicore-M5 on your M5Stack

Flash the latest Rudicore-M5 firmware directly from your browser. Use Google Chrome or Microsoft Edge on a desktop or laptop computer.

Installation steps

  • Make sure you have the required M5Stack USB drivers installed for your operating system. If not, install them from the official M5Stack USB driver page.
  • First, leave your M5Stack disconnected and click the "Connect" button below.
  • Your browser will show a popup dialog with a list of ports. Note which ports are shown.
  • Dismiss both dialogs (including the “No port selected” message) by clicking "Cancel".
  • Connect the M5Stack and click the "Connect" button below again to reopen the port list.
  • The new port that appears is usually your device. Typical names may include USB Serial, CP210x, CH340, or ttyUSB/ttyACM.
  • Select the newly appeared port, click "Connect" in the dialog, and confirm the installation.
  • Wait for flashing to complete. Keep the USB cable connected during the process. The device may reboot automatically when installation is finished.
Warning: Make sure no other ESP32 or related devices are connected because if you select the wrong port, you may accidentally flash those. This usually cannot be undone, so be careful.

Before you begin

  • Use a desktop browser
  • Use Chrome or Edge
  • Use a USB data cable
  • Disconnect other serial devices if possible

Troubleshooting

  • No port visible: reconnect the device and try a different cable.
  • Wrong port selected: unplug the M5Stack and see which port disappears.
  • Flashing fails: close other applications using the port and try again.
